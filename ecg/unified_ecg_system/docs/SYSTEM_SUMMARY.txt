## Usage Modes

### File Processing
```bash
python main_unified.py --file mydata
```
- Loads `mydata.ecg` and `mydata.ppg`
- Processes in 5-second chunks
- Displays in viewer
- Sends to backend

### Stream Recording
```bash
python main_unified.py --stream --duration 30 --port COM5
```
- Connects to HealthyPi on COM5
- Records for 30 seconds
- Same processing as file mode

### With AI Model
```bash
python main_unified.py --file mydata --model ./model_path
```
- Loads AI model
- Runs inference on each chunk
- Saves results to JSON

### No Backend
```bash
python main_unified.py --file mydata --no-backend
```
- Processes locally
- Skips backend transmission
- Useful for testing/debugging

### No Viewer
```bash
python main_unified.py --file mydata --no-viewer
```
- Batch processing mode
- No GUI required
- Useful for servers

## API Examples

### Load Signals
```python
from signal_loader import SignalFactory

# File
loader = SignalFactory.create('file', base_path='mydata')
ecg, ppg = loader.load()

# Stream
loader = SignalFactory.create('stream', duration=30)
ecg, ppg = loader.load()
```

### Process Signals
```python
from signal_sender import SignalProcessor

processor = SignalProcessor(sampling_rate=500)
results = processor.process_signals(
    {'I': ecg, 'II': ppg},
    send_to_backend=True
)
```

### Display Data
```python
from signal_viewer_pyqt import SignalViewer
from PyQt5.QtWidgets import QApplication

app = QApplication([])
viewer = SignalViewer(ecg_data=ecg, ppg_data=ppg)
viewer.show()
app.exec_()
```

## Architecture

```
CLI (main_unified.py)
    ↓
Signal Loading (signal_loader.py)
    ├─ File: FileSignalLoader
    └─ Stream: StreamSignalLoader → healthypi_lib.py
        ↓
Signal Processing (signal_sender.py)
    ├─ Normalize
    ├─ FFT/MFCC/Spectrum
    ├─ AI Model (optional)
    └─ Backend API (optional)
        ↓
Visualization (signal_viewer_pyqt.py)
    └─ PyQt5 Display + Statistics
```

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Import error | `pip install -r requirements_unified.txt` |
| File not found | Check file path and extension (.ecg, .ppg) |
| Serial port error | Use `--port` flag, check device |
| Viewer won't open | Ensure display available, check PyQt5 |
| Backend fails | Use `--no-backend`, check credentials |
